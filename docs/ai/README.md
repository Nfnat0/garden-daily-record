# AI資料インデックス

このフォルダは、Garden Daily Record をAIと人間が継続的に改善するための作業資料です。

## 資料一覧

- `repository-guide.md`: プロダクト概要、構成、主要ファイルの責務。
- `development-workflow.md`: 起動、ビルド、テスト、ブラウザ検証の手順。
- `coding-standards.md`: 実装規約、i18n、生成物、デザイン上の注意点。
- `ux-improvement-plan.md`: 現状UXの評価と改善ロードマップ。
- `procedures.md`: UXレビュー、実装、検証、ドキュメント更新の手順。
- `progress.md`: 進捗、判断、次アクションの管理表。

## 運用ルール

- UXタスクの状態が変わったら `progress.md` を更新する。
- 新しい調査で優先度が変わったら `ux-improvement-plan.md` を更新する。
- 構成やコマンドが変わったら `repository-guide.md`、`development-workflow.md`、`coding-standards.md` を更新する。
- `AGENTS.md` は入口として短く保ち、詳細は `docs/ai/` に置く。
